# 1. Condicional 
Condicional pode ser entendido como uma escolha. Para fazer essas escolhas usamos os operadores logicos ou matematicos.

Exemplo:
- +, -, *, /
- <, >, =
- _and, on, not_

As palavras reservadas para isso sâo: **if**, **else**, **else if**
## If
A condição **if** significa "se". Se atender as condições faça alguma coisa
## Else
A condição **else** significa "senao", quer dizer que a primeira condição não foi aceita e agora estamos no else
## Else If
A condição "senao se". Essa condição não pode ser utilizada nem a primeira nem a última condição pois representa como se fosse uma condição intermediaria

# 2. Operador ternário

Essa operação é bem parecida com **if else**, contudo usaremos os  interrogação(?) e dois pontos(:)
